## Installation

Kore.kt is available via the public Maven repository:

**Repository:**  
[@repo.jespersen.zip:zip.jespersen.korekt](https://repo.jespersen.zip/#/releases/zip/jespersen/Kore.kt)

Add it to your `build.gradle.kts`:

```
maven {
    name = "repo.jespersen.zip"
    url = uri("https://repo.jespersen.zip/releases")
}
```
